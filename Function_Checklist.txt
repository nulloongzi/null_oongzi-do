1. 🐍 데이터 처리 (Python Logic)
[x] 구글 시트 연동: 최신 CSV URL 적용 (...output=csv)

[x] 데이터 파싱 규칙:

I열(이메일) 건너뜀.

J열(9번): 'O' 표시 확인 → 긴급 구인 여부 판단.

K열(10번): 긴급 구인 메시지 추출.

[x] 좌표 변환 (Geocoding): 카카오 API로 주소를 좌표(lat, lng)로 변환.

[x] Jittering (겹침 방지): 같은 주소(같은 체육관)를 쓰는 팀들은 좌표를 미세하게 벌려서 겹치지 않게 배치.

[x] 테스트 모드 스위치: IS_TEST_MODE = True 설정 시 test_new.html 생성 (캐시 문제 해결용).

[x] 지도 중심 설정: 'GVT' 팀의 좌표를 찾아 지도의 초기 중심점으로 설정.

2. 📍 마커 & 지도 표시 (Map & Marker)
[x] 커스텀 디자인:

일반: 노란색 배구공 (marker_yellow.png).

긴급: 빨간색 배구공 (marker_red.png).

[x] 긴급 구인 강조 로직:

색상: 빨간 마커 사용.

우선순위: zIndex: 9999 적용 (다른 마커보다 무조건 위에 뜸).

클러스터링 제외: 지도를 축소해도 묶이지 않고 독자적으로 떠 있음.

라벨(이름표): '🔥' 아이콘 포함 + 항상 보임 + Pulse(두근두근) 애니메이션.

[x] 일반 마커 로직:

클러스터링: 지도를 축소하면 노란색 원으로 묶임.

라벨: 지도 레벨 5 이하(확대했을 때)에서만 이름표가 나타남.

[x] 라벨 위치: 마커 이미지 높이(53px)를 고려하여 마커 머리 위에 뜨도록 CSS 조정 (translateY(-55px)).

3. 📱 UI/UX 및 인터랙션 (Interaction)
[x] 마커 클릭 액션:

바텀시트 오픈: 팀 상세 정보 표시.

자동 확대: 지도 레벨을 4로 확대.

중앙 정렬(Offset): 바텀시트에 가려지지 않도록 화면 중앙에서 살짝 위(0.13 비율)로 부드럽게 이동(panTo).

[x] 검색 & 필터:

검색창: 팀명, 지역명 검색 가능.

필터 시트: 지역, 요일, 대상(성인/대학생 등) 다중 선택 가능.

자동 범위 재설정: 검색 결과가 나오면 그 팀들이 다 보이도록 지도 범위(bounds) 자동 조절.

[x] 내 위치 찾기: 우측 하단 '📍' 버튼 클릭 시 사용자 현재 위치로 지도 이동.

[x] 새로고침: 우측 하단 '🔄' 버튼 제공.
