✅ 누룽지도 최종 기능 명세서 (2025.01 Final Ver.)
1. 📲 모바일 최적화 & PWA (New!)
[x] 앱 모드 설정: 홈 화면에 추가 시 주소창 없이 앱처럼 실행 (manifest.json, mobile-web-app-capable).

[x] 테마 컬러: 상단바 색상을 흰색(#ffffff)으로 고정.

[x] 터치 인터랙션:

필터 시트 드래그: 필터 창 하단 핸들을 잡고 내리면 닫힘.

바텀 시트 드래그: 상세 정보 창 핸들을 잡고 내리면 닫힘.

[x] 키보드 자동 숨김: 검색이나 마커 클릭 시 모바일 키보드가 자동으로 내려감 (blur).

2. 📍 마커 & 지도 표시 (Map & Marker)
[x] 커스텀 디자인:

일반: 노란색 배구공 (marker_yellow.png).

긴급: 빨간색 배구공 (marker_red.png).

[x] 긴급 구인 강조 로직:

우선순위: zIndex: 9999 (항상 맨 위).

클러스터 제외: 묶이지 않고 지도에 바로 표시.

라벨 효과: 🔥 아이콘 + Pulse(두근두근) 애니메이션.

[x] 일반 마커 로직:

클러스터링: 축소 시 노란색 원으로 그룹화.

라벨: 확대 레벨 5 이하에서만 이름표 노출.

[x] 라벨 위치 보정: 새 마커 높이에 맞춰 translateY(-55px)로 상향 조정.

3. 📱 UI/UX 및 인터랙션
[x] 마커 클릭 액션:

자동 확대: 지도 레벨 4로 진입.

시야 확보(Offset): 바텀시트 높이를 고려해 화면 중앙에서 **13% 위쪽(0.13)**으로 부드럽게 이동.

[x] 스마트 검색 & 필터:

통합 검색: 팀명, 지역명 검색.

상세 필터: 지역/요일/대상 다중 선택.

자동 시야 조절: 검색 결과가 한눈에 들어오도록 지도 범위(bounds) 자동 맞춤.

조건부 닫기: 필터 적용 버튼을 누를 때만 창이 닫힘 (오작동 방지).

[x] 편의 기능:

내 위치 찾기: 우측 하단 📍 버튼.

새로고침: 우측 하단 🔄 버튼 (fab-group).

4. 📄 바텀시트 (Rich UI)
[x] 디자인 복구: 아이콘, 태그, 버튼이 포함된 풍성한 레이아웃.

[x] 정보 표시: 팀 이름, 스케줄, 회비, 인스타 링크(아이콘).

[x] 긴급 배너: 긴급 구인 시 상단에 빨간색 띠 배너(메시지 포함) 노출.

[x] 액션 버튼:

📍 주소 복사 (클립보드).

🚀 길찾기 (카카오맵 연동).

5. 🐍 데이터 처리 (Back-end Logic)
[x] 구글 시트 연동: 최신 CSV 자동 파싱.

[x] 데이터 가공: I열 제외, J열(긴급여부)/K열(메시지) 처리.

[x] 지리 정보: 주소 좌표 변환 & Jittering(겹침 방지) 적용.

[x] 초기 설정: 지도 중심을 'GVT' 팀 좌표로 자동 설정.
